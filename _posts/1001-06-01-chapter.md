#### Docker Registry

Tıpkı github gibi geliştiriciler açık kaynaklı olarak docker imagelerini yükleyerek ve [DockerHub](https://hub.docker.com/)’ta paylaşarak imagelerin bizim de indirip kullanmamıza olanak sağlıyorlar. Kısaca imagelar Docker Registrylerde tutuluyor. Örneğin siz postgres  image’ını kullanmak istiyorsunuz; [postgres image](https://hub.docker.com/_/postgres) linkinden

`docker pull postgres`

komutu ile indirip artık bu image ile container oluşturabiliyorsunuz.
